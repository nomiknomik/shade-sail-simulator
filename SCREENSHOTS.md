# 📸 Screenshots hinzufügen

Das README zeigt Bilder aus dem `screenshots/`-Ordner.  
So fügst du echte Screenshots hinzu:

## Empfohlene Screenshots

| Dateiname | Inhalt |
|---|---|
| `screenshots/3d-view.png` | 3D-Ansicht mit Segel + Schattenwurf (ca. 1200×700 px) |
| `screenshots/panel.png` | Linkes Steuerungspanel (ca. 400×900 px) |
| `screenshots/fullscreen.png` | Vollbild App-Ansicht (1920×1080 px) |

## Anleitung

1. App öffnen: https://nomiknomik.github.io/shade-sail-simulator/
2. Screenshot mit gewünschter Szene machen (z.B. Sommermittag mit Segel)
3. Als PNG in den `screenshots/`-Ordner committen

```bash
git add screenshots/3d-view.png
git commit -m "docs: screenshot hinzugefügt"
git push
```

Bis Screenshots vorhanden sind, zeigt das README broken image links (nur im Markdown).
